# galtonboard-qt

This is a galton board simulator on Qt using box2d as a physics engine.

# Environment

- Qt Creator: 8.0.2
- Qt: 6.4.0
- Confirmed Compiler:
    - Desktop Qt 6.4.0 MSVC2019 64bit
    - Android Qt 6.4.0 Clang arm64-v8a
